# ⚠️ MEGA Network Issue Detected

## Problem

MEGA API is not accessible from your network:
```
ReadTimeoutError: HTTPSConnectionPool(host='g.api.mega.co.nz', port=443): Read timed out
```

## Possible Causes

1. **Firewall/Antivirus blocking MEGA**
2. **ISP blocking MEGA** (some ISPs block file-sharing services)
3. **Network restrictions** (corporate/school network)
4. **MEGA servers slow/down** in your region

## Solutions

### Option 1: Use VPN (Recommended for MEGA)
1. Install a VPN (ProtonVPN, Windscribe - both have free tiers)
2. Connect to VPN
3. Restart server
4. MEGA should work
